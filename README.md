# 🎬 Movie Review Sentiment Analyzer

A Machine Learning web application that predicts whether a movie review is positive or negative using Natural Language Processing (NLP).

## Features

* Sentiment classification of movie reviews
* TF-IDF text vectorization
* Logistic Regression classifier
* Confidence score for predictions
* Interactive Streamlit interface

## Tech Stack

* Python
* Scikit-Learn
* Pandas
* Streamlit
* Joblib

## Project Workflow

Review Text → TF-IDF Vectorizer → Logistic Regression → Sentiment Prediction

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Example

Input:
The movie was absolutely fantastic and enjoyable.

Output:
Positive 😊
